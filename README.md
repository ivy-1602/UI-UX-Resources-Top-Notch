## 📖 About

This project is a single-page resource hub for designers, creators, and learners who want to understand how people think, interact, and connect through design.

The resources are organized into three levels:
- **Beginner** — Fundamentals, Figma basics, UX process
- **Intermediate** — Advanced workflows, video editing, UI libraries
- **Pro** — Professional design systems, interface patterns, comprehensive courses

No distractions. No noise. Just curated, high-quality content to help you grow at your own pace.

---

## ✨ Features

- **Clean, gradient-based design** with a soft, modern aesthetic
- **Organized by skill level** so you can start where you are
- **Diverse resource types**: PDFs, videos, articles, GitHub repos, courses
- **Fully responsive** — works beautifully on mobile and desktop
- **Carefully reviewed content** — every link has been tested and verified

## 🛠️ Built With

- **HTML5** — Semantic structure
